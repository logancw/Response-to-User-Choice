## Assignment  Goal: 
This lab was designed to teach you how to design and use classes, instantiate objects, and use switch cases.

## Project Description: 
Write a program in Java that establishes a range of data such as responses to user choice.  The computer should navigate the group of data and possible outputs using switch cases. The user choice needs to be made in a Runner class that passes parameters to the sub class to run the method containing switch classes. 
The user should have the option to pick from at least 10 different choices and your method should output a unique response to each of the 10 user choices.
